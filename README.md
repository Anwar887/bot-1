## Required
-   nodejs
-   git

## Tutorial
1. Cloning this repository<br />
```sh
git clone https://github.com/FERDIZ-afk/base-bot-whatsapp.git
cd base-bot-whatsapp
```
<br />

2. Install dependencies<br />
```sh
npm i
```
<br />

3. Run Bot<br />
```sh
npm start
```
<br />

5. untuk session di simpan di folder (/save/session.json)<br />
